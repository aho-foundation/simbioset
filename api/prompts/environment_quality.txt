КОНЦЕПЦИЯ:
- Состояние окружающей среды определяет возможности для симбиозов
- Климатические условия влияют на жизнедеятельность организмов
- Качество среды обитания определяет потенциал для улучшения экосистем

ИНСТРУКЦИИ:
1. Извлеки климатические условия:
   - Температура (приблизительная, если видно по признакам: снег, листва, активность организмов)
   - Влажность (высокая/средняя/низкая по признакам: туман, роса, сухость)
   - Осадки (дождь, снег, засуха - по видимым признакам)
   - Ветер (сильный/слабый по признакам: движение листвы, волны)
   - Сезон (весна/лето/осень/зима по признакам: листва, цветение, снег)
   - Освещенность (солнечно/пасмурно/сумерки)

2. Определи состояние окружающей среды:
   - Качество воздуха (по видимости, наличию дыма, тумана, прозрачности)
   - Качество воды (если видна вода: прозрачность, цвет, наличие пены/загрязнений)
   - Состояние почвы (видимое: влажная/сухая, плодородная/бедная, эрозия)
   - Уровень загрязнения (видимые признаки: мусор, отходы, неестественные цвета)
   - Антропогенное воздействие (дороги, здания, следы деятельности человека)

3. Извлеки полезные данные для поиска симбиозов:
   - Тип биома (лес, степь, тундра, пустыня, водная среда и т.д.)
   - Доступность ресурсов (вода, питательные вещества, свет, укрытия)
   - Угрозы для экосистемы (засуха, загрязнение, вырубка, эрозия)
   - Потенциал для улучшения (что можно улучшить в среде обитания)
   - Благоприятные условия для симбиозов (наличие разных видов, разнообразие)

4. Оцени общее состояние:
   - Здоровье экосистемы (здоровая/нарушенная/деградированная)
   - Биоразнообразие (высокое/среднее/низкое по видимым признакам)
   - Стабильность (стабильная/нестабильная по признакам нарушений)
