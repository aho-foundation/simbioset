# ИНСТРУКЦИИ ПО ВЫДЕЛЕНИЮ АРТЕФАКТОВ

1. **Анализ диалога**:
   - Прочитай весь диалог внимательно
   - Выдели ключевые темы и проблемы
   - Найди предложения решений, идеи, гипотезы
   - Отметь важные факты, наблюдения, вопросы

2. **Критерии значимости артефакта**:
   - Конкретность: можно ли реализовать или проверить?
   - Ценность: полезно ли для проекта Simbioset?
   - Новизна: содержит ли новую информацию или идею?
   - Потенциал: можно ли развить в полноценный проект?

3. **Типы артефактов**:
   - idea (идея): творческое предложение или концепция
   - requirement (требование): необходимое условие или функциональность
   - problem (проблема): выявленная задача или препятствие
   - solution (решение): предложенный способ решения проблемы
   - insight (озарение): важное понимание или наблюдение
   - hypothesis (гипотеза): предположение, требующее проверки

4. **Качество артефактов**:
   - Длина: 1-3 предложения (не слишком длинно, но достаточно конкретно)
   - Язык: четкий, технический, без лишней воды
   - Контекст: сохранить связь с исходным сообщением
   - Приоритет: сначала самые ценные и конкретные артефакты

# ФОРМАТ ОТВЕТА

Верни ТОЛЬКО JSON объект следующего формата:

```json
{
  "artifacts": [
    {
      "id": "artifact_1",
      "type": "idea|requirement|problem|solution|insight|hypothesis",
      "content": "Краткое описание артефакта (1-3 предложения)",
      "original_text": "Цитата из диалога",
      "message_index": 3,
      "confidence": 0.85,
      "tags": ["экосистема", "симбиоз", "оптимизация"],
      "rationale": "Почему этот артефакт значим и ценен"
    }
  ],
  "summary": {
    "total_artifacts": 5,
    "main_themes": ["симбиотические отношения", "экосистемная оптимизация"],
    "project_potential": "высокий|средний|низкий",
    "recommendations": ["Создать прототип системы анализа симбиоза"]
  }
}
```

ПРАВИЛА:
- id должен быть уникальным (artifact_1, artifact_2, etc.)
- confidence: оценка уверенности в значимости (0.0-1.0)
- tags: 2-5 релевантных тегов
- rationale: краткое объяснение ценности (1-2 предложения)
- summary: общая оценка диалога
