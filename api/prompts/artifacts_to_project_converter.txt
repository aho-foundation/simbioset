- Опенсорсные датасеты: использование открытых биологических/экологических данных

# КОНТЕКСТ ПРОЕКТА SIMBIOSET
Основной проект: Simbioset (биологическая оптимизация экосистем)
Цели: Понимание и улучшение биологических систем через симбиотические отношения
Подход: Краудфандинг + Краудсорсинг + Опенсорсные датасеты
Технологии: ИИ, машинное обучение, биоинформатика, экосистемный анализ
Ресурсы: Опенсорсные датасеты (GBIF, iNaturalist, Open Tree of Life, citizen science данные)

# ИНСТРУКЦИИ ПО СОЗДАНИЮ ПРОЕКТОВ

1. **Анализ артефактов**:
   - Изучите все предоставленные артефакты
   - Найдите связи и зависимости между ними
   - Группируйте связанные артефакты в логические проекты
   - Определите приоритеты и зависимости

2. **Критерии формирования проекта**:
   - Связность: артефакты должны логически объединяться
   - Реалистичность: проект должен быть выполнимым за разумное время
   - Ценность: результат должен быть полезным для Simbioset
   - Масштабируемость: проект можно расширить или повторить

3. **Типы проектов**:
   - crowdfunded: КРАУДФАНДИНГОВЫЕ проекты (ВСЕГДА этот тип для проектов из артефактов)
     * Финансирование через краудфандинговые платформы
     * Краудсорсинг для сбора данных и тестирования
     * Использование опенсорсных датасетов
     * Сообщество помогает в разработке и валидации

4. **Структура КРАУДФАНДИНГОВОГО проекта**:
   - Название: четкое и запоминающееся для привлечения спонсоров
   - Описание: убедительная история + научная ценность + impact
   - Краудфандинг цель: конкретная сумма для реализации MVP
   - Периоды: сбор средств (1-3 месяца) → разработка (3-6 месяцев) → запуск
   - Краудсорсинг компоненты: сбор данных сообществом, тестирование, валидация
   - Опенсорсные датасеты: какие открытые данные будут использоваться
   - Ресурсы: серверы, API ключи, датасеты, инструменты разработки
   - Метрики успеха: научные показатели + пользовательские метрики

5. **Артефакты в проект**:
   - Преобразуйте артефакты в конкретные задачи или требования
   - Создайте иерархию: цели → задачи → подзадачи
   - Определите ответственных и сроки
