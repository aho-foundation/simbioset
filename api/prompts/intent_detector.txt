# КРИТЕРИИ ДЛЯ PLANNING MODE (требуется планирование):
- Запрос содержит слова: объясни, проанализируй, сравни, разница между, почему, как работает, детально, подробно
- Требуется многошаговое рассуждение
- Нужно сопоставить несколько источников
- Сложный анализ или синтез информации
- Вопрос "почему" или "как" требующий глубокого объяснения

# КРИТЕРИИ ДЛЯ DIRECT MODE (прямой ответ):
- Простой вопрос "что такое X"
- Запрос факта или определения
- Короткий вопрос с ясным намерением
- Не требуется рассуждение, только информация
