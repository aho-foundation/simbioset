# Функциональность Simbioset

## Артефакты беседы (v0.4.19)

### Обзор
Система артефактов позволяет пользователям выделять важные фрагменты текста в сообщениях чата и создавать из них краудфандинговые проекты с использованием краудсорсинга и опенсорсных датасетов.

### Возможности
- **Выделение текста**: Выделите любой текст в сообщении для работы с ним
- **Маркировка артефактов**: Кнопка ⭐ в действиях сообщения для сохранения выделенного текста
- **Управление артефактами**: Панель просмотра всех отмеченных артефактов с возможностью редактирования и удаления
- **Создание краудфандинговых проектов**: Автоматическая генерация проектов с целевыми суммами финансирования
- **Краудсорсинг компоненты**: Участие сообщества в сборе данных, тестировании и улучшении
- **Опенсорсные датасеты**: Интеграция с GBIF, iNaturalist, Open Tree of Life и другими
- **Контекстные действия**: Все действия (копирование, фактчек, поиск) работают с выделенным текстом

### Использование
1. Выделите текст в любом сообщении чата
2. Нажмите на иконку ⭐ в панели действий сообщения
3. Артефакт будет сохранен в сессии
4. Откройте панель артефактов кнопкой "Артефакты" в нижней панели
5. Создайте проект из артефактов или управляйте ими

### Промпты для ИИ
- `dialog_artifacts_extractor.txt` - извлечение артефактов из диалога
- `artifacts_to_project_converter.txt` - преобразование артефактов в проекты

### API
- `POST /api/artifacts` - создание артефакта
- `GET /api/artifacts` - получение артефактов сессии
- `DELETE /api/artifacts/{id}` - удаление артефакта
- `POST /api/artifacts/analyze-conversation` - анализ диалога и извлечение артефактов
- `POST /api/artifacts/create-project-from-conversation` - полный цикл создания проекта из диалога
- `POST /api/artifacts/convert-artifacts-to-projects` - преобразование артефактов в проекты
- `POST /api/projects/from-artifacts` - создание проекта из артефактов

## Операции выбора узлов (v0.4.18)

### Обзор
Система выбора позволяет отмечать узлы в дереве знаний для выполнения операций над группами узлов.

### Возможности
- Выбор отдельных узлов
- Переключение состояния выбора
- Получение списка выбранных узлов
- Очистка всего выбора
- Визуальные индикаторы выбранных узлов

### API
- `POST /api/kb/nodes/{node_id}/select` - выбор узла
- `POST /api/kb/nodes/{node_id}/toggle` - переключение выбора
- `GET /api/kb/nodes/selected` - получение выбранных узлов
- `POST /api/kb/nodes/clear-selection` - очистка выбора
